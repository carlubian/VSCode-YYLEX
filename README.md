# YYLEX Extension

Provides support for yylex files within VS Code.

## Features

* Syntax coloring and token name highlighting.
* Automatic brace insertion

## Known Issues

When writing a '%{' brace, a single '}' will be inserted, instead of the corresponding '%}'.
